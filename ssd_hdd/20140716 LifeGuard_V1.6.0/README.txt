Утилита для для просмотра неизвестных(для smartctl) атрибутов ssd дисков Apacer

скрипты start.* находят подключенный к сисетме ssd и выводят на экран его smart инфу

Important notice:

Be careful while selecting disk destination, in some cases choosing another disk may cause 

data get corrupt and OS stop booting!

Application is built for 32bit in case Linux displays error No such file or directory?

You must add 32bit support running command for Ubuntu 13.10 & 14.04 sudo apt-get install 

-y lib32z1 lib32ncurses5 lib32bz2-1.0

I case Linux display error while loading shared libraries: libstdc++.so.6: cannot open shared 

object file: No such file or directory

Try adding apt-get install lib32stdc++6
